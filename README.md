## Overview

웹 개발과 AI 기술을 융합하여 실질적인 서비스를 만드는 개발자, 허준영의 포트폴리오입니다.

### Features

- **3D Interactive Hero** — Three.js 기반 인터랙티브 3D 오브젝트
- **Dark / Light Theme** — 다크 모드 기본, 라이트 모드 전환 지원
- **Bilingual** — 한국어 / English 자동 감지 및 수동 전환
- **Scroll Animations** — Framer Motion 기반 스크롤 애니메이션
- **Glassmorphism UI** — 반투명 글래스모피즘 카드 디자인
- **3D Card Tilt** — 마우스 추적 3D 틸트 효과
- **Typing Animation** — 역할 자동 타이핑 효과
- **Responsive** — 데스크탑 / 태블릿 / 모바일 완벽 대응
